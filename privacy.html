Privacy Policy / Política de Privacidade

Last updated / Última atualização: February 2026

1. Purpose / Finalidade

AclTime Pro is an Android application developed exclusively for educational, scientific, and exploratory purposes.

O AclTime Pro é um aplicativo Android desenvolvido exclusivamente para fins educacionais, científicos e exploratórios.

2. Information Collection / Coleta de Informações

AclTime Pro does not require user registration and does not intentionally collect personally identifiable information.

O AclTime Pro não exige cadastro de usuários e não coleta intencionalmente informações pessoais identificáveis.

3. Data Usage / Uso dos Dados

Any data voluntarily entered by the user is processed locally on the device and used solely for estimation and simulation purposes.

Quaisquer dados inseridos voluntariamente pelo usuário são processados localmente no dispositivo e utilizados apenas para fins de simulação e estimativa.

4. Data Storage / Armazenamento

AclTime Pro does not transmit personal data to external servers.

O AclTime Pro não transmite dados pessoais para servidores externos.

5. No Medical or Legal Use / Não Uso Médico ou Legal

The application does not provide medical advice, diagnosis, treatment, or legal determinations.

O aplicativo não fornece aconselhamento médico, diagnóstico, tratamento ou determinações legais.

6. Changes / Alterações

This Privacy Policy may be updated at any time as the project evolves.

Esta Política de Privacidade pode ser atualizada a qualquer momento, conforme a evolução do projeto.
